# 常用命令

#### java

##### 命令格式

```json
java [-option] -jar
```

##### option参数

```json
-server, to select the "server VM", is default

-D<name>=<value>, set a system property
```

##### 示例

```json
java -Dspring.profile.active=pro -jar xxx.jar
```
