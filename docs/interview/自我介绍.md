# 自我介绍

## 关于面试

  我认为，面试要先声夺人。

  就是最起码，你的回答不能是大通货，如果所有人都这么回答，你也这么回答，那你怎么保证自己比其他面试者强。所以一定要有自己的见解，要答出别人不知道或者没想到的答案。

  然后，你的回答最好能把面试官带到你擅长的领域。
